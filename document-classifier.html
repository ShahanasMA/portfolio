What & Why

With the rapid growth of digital information, managing and organizing large volumes of documents has become a significant challenge. Manual document classification is time-consuming, error-prone, and inefficient for large-scale repositories.

This system proposes an Intelligent Document Classification System that automatically categorizes documents based on their textual content using ML techniques.

The system applies NLP preprocessing (tokenization, stop-word removal, text normalization) followed by TF-IDF feature extraction and SVM classification to predict document categories with high accuracy.

Beyond classification, the system includes an admin review mechanism, notification system, and user authentication for a complete, production-ready document management platform.

What It Does
Automated Classification
Automatically categorizes documents into HR, Finance, Legal, and Administration using SVM machine learning model.
NLP Text Processing
Tokenization, stop-word removal, text normalization, and TF-IDF feature extraction for accurate text analysis.
Confidence Score
Each classification result includes a confidence score to indicate prediction reliability.
Notification System
Real-time admin notifications for pending document reviews and low-confidence predictions.
Admin Review Panel
Administrators can review, approve, or correct document classifications flagged for review.
User Authentication
Secure login, registration, and admin approval system with role-based access control.
Multi-format Support
Supports PDF, TXT, and PPTX document uploads with automatic text extraction.
Correction Requests
Users can submit correction requests for misclassified documents, which are reviewed by admins.
Real-time Processing
Documents are classified within 1–2 seconds, providing near real-time results to users.
